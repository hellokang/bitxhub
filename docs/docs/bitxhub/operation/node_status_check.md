# 节点状态检查

这里主要介绍如何检查跨链系统中bitxhub和pier节点的进程状态。

### BitXHub

可以通过ps命令查看节点进程的运行状态，示例如下：

`ps aux|grep bitxhub`

正常情况下可以看到bitxhub节点的进程存活。

### Pier

可以通过ps命令查看节点进程的运行状态，示例如下：

`ps aux|grep pier`
